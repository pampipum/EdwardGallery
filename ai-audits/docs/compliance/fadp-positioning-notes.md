# FADP / nDSG Positioning Notes

These notes support messaging and process design. They are not legal advice.

## Core Principles To Reference

Based on official Swiss FDPIC guidance and the revised Federal Act on Data Protection:

- transparency when collecting and processing personal data
- clear purpose specification
- proportionality / data minimization
- secure processing
- access and deletion expectations

## Practical Messaging For The Offer

- We tell clients what data is needed and why.
- We ask for the minimum necessary inputs.
- We prefer reduced, redacted, or anonymized examples where possible.
- We document where transcript and analysis data are processed.
- We define deletion expectations after the engagement.

## Delivery Guardrails

- avoid uploading unnecessary personal data into third-party tools
- separate sensitive medical or client-identifying data from general workflow analysis
- prefer Swiss or EU data residency when tooling allows
- keep a human review step before recommendations are finalized
