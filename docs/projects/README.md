# Project Briefs

Store one file per project here.

Naming:
- use kebab-case
- example: `ai-audits.md`, `landing-page-redesign.md`, `alphaarena-migration.md`

Each project file should define:
- goal
- PM / owner
- project rules
- update format
- deliverables
- notes / links / session keys

Default reporting format:
- Status
- Blockers
- Next steps
